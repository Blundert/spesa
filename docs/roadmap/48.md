# Task 48 — Variazione Prezzo in Spesa (v0.16.0)

## Cosa è stato fatto

Aggiunto un indicatore di variazione prezzo nel `PriceKeypad`, visibile in tempo reale mentre l'utente digita il prezzo di un articolo durante la spesa.

## Modifiche

- **`src/components/PriceKeypad.tsx`**: aggiunto prop `lastPriceCents?: number | null` e componente interno `PriceCompareRow` che mostra sotto il display del prezzo:
  - `null` → "primo acquisto" (grigio muted)
  - `diff === 0` → "stesso prezzo dell'ultima volta" (grigio)
  - `diff > 0` → "↑ +€X,XX rispetto all'ultima volta (€Y,YY)" (rosso `#D14343`)
  - `diff < 0` → "↓ −€X,XX rispetto all'ultima volta (€Y,YY)" (verde `#2E9E6B`)
- **`src/screens/SpesaScreen.tsx`**: aggiunto `lastPriceCents` allo state `priceTarget`, costruita `itemLastPriceMap` dagli item caricati, passato `lastPriceCents` al `PriceKeypad` delle voci lista e carrello.
- **`src/i18n/locales/it.ts`** e **`en.ts`**: aggiunte chiavi `priceFirstBuy`, `priceSame`, `priceHigher`, `priceLower`.

## Scelte non banali

- Non mostrato l'indicatore per il keypad "oggetto fuori lista" (nuovo item, nessun prezzo storico disponibile al momento dell'inserimento).
- Per la modifica di un acquisto già nel carrello si mostra comunque l'indicatore usando `lastPriceCents` dall'item — utile come riferimento storico anche in quel contesto.
