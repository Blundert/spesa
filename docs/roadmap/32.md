## Passi manuali (utente)
1. Crea repo pubblico `blundert/spesa` (nome libero).
2. `git remote add origin https://github.com/blundert/spesa.git` → `git push -u origin main`.
3. Settings → Pages → Source = "GitHub Actions".
4. Settings → Pages → Custom domain = `spesa.matteogranzotto.com` → Save → poi Enforce HTTPS.
5. DNS di mattegranzotto.com: record CNAME `spesa` → `blundert.github.io.`.
