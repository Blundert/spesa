**Data:** 2026-06-15

## File creati

### `src/lib/money.ts`
- `toCents(euros)` — converte decimale → centesimi interi (Math.round)
- `fromCents(cents)` — centesimi → decimale
- `formatCents(cents)` — "X,XX" con simbolo € (locale italiano)
- `formatCentsPlain(cents)` — "X,XX" senza simbolo
- `addCents(a, b)` — somma sicura
- `clampCents(cents)` — min 0

Tutti gli importi nel codebase sono interi in centesimi. Mai float per calcoli monetari.

### `src/lib/date.ts`
- `getISOWeek(date)` — numero settimana ISO 8601
- `toISOWeekString(date)` → "2026-W24" (gestisce boundary year-end)
- `currentISOWeek()` — week corrente
- `weekRange(isoWeek)` → `{ monday, sunday }` come Date UTC
- `dayShort(idx)` → "Lun"…"Dom"
- `dayFull(idx)` → "Lunedì"…"Domenica"
- `formatWeekLabel(isoWeek)` → "9 – 15 giu"
- `formatShortDate(ts)` → "7 giu"
- `formatLongDate(ts)` → "lun 9 giu 2026"
