# Task 46 — Categorie in ordine alfabetico (v0.14.6)

## Problema

Le categorie nel Catalogo (lista, gruppi articoli, picker "cambia categoria") e nella Lista
apparivano in ordine di inserimento nel DB (sortOrder), non alfabetico.

## Root cause

`getCategories()` in `src/db/repositories/categories.ts` usava `orderBy('sortOrder')`.
Il campo `sortOrder` riflette l'ordine di creazione, non un ordine semantico.

## Fix

Cambiato `getCategories()` per ordinare per nome con `localeCompare` italiano case-insensitive.
Il campo `sortOrder` è mantenuto invariato (ancora usato da `categoryLabel` per le seed 0–5).
