# Task 49 — Quantità Modificabile nella Lista (v0.17.0)

## Cosa è stato fatto

Aggiunto uno stepper inline (− / quantità / +) ad ogni articolo nella ListaScreen per modificare la quantità direttamente dalla lista, senza dover aspettare la spesa.

## Modifiche

- **`src/screens/ListaScreen.tsx`**:
  - Importato `useUpdateListQuantity` da `useListItems`.
  - Il cestino è stato rimosso: il pulsante `−` a quantità 1 apre il dialog di conferma rimozione già esistente (`setPendingListRemove`).
  - Il pulsante `+` chiama `updateListQuantity.mutate({ id, quantity: li.quantity + 1 })`.
  - Il layout della riga è ora: `[−] [qty] [+]  [nome/prezzo]  [→ dettaglio]`.

## Scelte non banali

- `min-w-0` + `truncate` aggiunti al div del nome per evitare overflow orizzontale con nomi lunghi.
- Il cestino separato è stato rimosso per mantenere il layout compatto; `−` a qty=1 copre quel caso con il flusso di conferma esistente.
