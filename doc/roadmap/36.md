# Task 36 — Fix aggiornamento PWA + Schermata Changelog (v0.8.0)

## Cosa è stato fatto

### Fix aggiornamento PWA
- `handleUpdate` in `ImpostazioniScreen` ora de-registra tutti i service worker e cancella tutte le cache prima di ricaricare (approccio "nuclear"), eliminando la race condition in cui il reload scattava prima che il nuovo SW prendesse controllo.
- Aggiunto `cleanupOutdatedCaches: true` in `vite.config.ts` → i bucket di cache delle versioni precedenti vengono eliminati automaticamente ad ogni attivazione di un nuovo SW.

### Schermata Changelog
- Nuovo file `src/data/changelog.ts`: dizionario `version → { it, en }` con bullet list per ogni versione, dalla 0.6.4 alla 0.8.0.
- Nuova schermata `src/screens/ChangelogScreen.tsx`: lista versioni in ordine decrescente, badge "Attuale/Current" sulla versione corrente, localizzata IT/EN.
- Route standalone `/changelog` aggiunta in `src/router.ts`.
- Riga "Versione" in `ImpostazioniScreen` trasformata in bottone cliccabile che naviga a `/changelog`, con chevron destro.
- Nuove chiavi i18n `settings.changelog` e `settings.current` in `it.ts` e `en.ts`.
