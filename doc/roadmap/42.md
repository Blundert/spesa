# Task 42 — Fix bordo bianco icona iOS

## Problema

Dopo il cambio del background dell'icona da bianco a beige (`#F2F2F0`), sull'home screen iOS appariva un bordo bianco intorno all'icona Spesa.

## Causa

`minimal2023Preset` applica un padding del 30% all'`apple-touch-icon`: genera un canvas 180×180 con sfondo bianco e l'icona scalata a 126×126 al centro → 27px di bordo bianco su ogni lato. Il parametro `padding: 0.5` già presente nel config si applica solo agli splash screen, non alle icone app.

## Soluzione

In `pwa-assets.config.ts`, spread del preset con override `apple: { sizes: [180], padding: 0 }`. Con padding zero e background beige nell'SVG, il canvas viene riempito completamente senza bordi.

## File modificati

- `pwa-assets.config.ts` — aggiunta `apple: { sizes: [180], padding: 0 }`
- `public/apple-touch-icon-180x180.png` — rigenerato (beige pieno, nessun bordo)
- Tutti i file in `public/` — rigenerati con `pwa-assets-generator`
- `package.json` — versione → 0.14.2
