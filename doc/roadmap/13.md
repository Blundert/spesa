# Task 13 — Modalità spesa (slice verticale)

**Data:** 2026-06-16
**Commit:** `feat: shopping mode screen (vertical slice)`

## Implementato in `src/screens/SpesaScreen.tsx`

### Flusso completo
1. Se nessuna sessione attiva → schermata "scegli supermercato" con `StorePickerSheet`
2. Sessione attiva → schermata spesa con hero budget, lista da prendere, carrello
3. Tap su item da lista → `PriceKeypad` → acquisto aggiunto + toast con rimanente
4. FAB "Aggiungi fuori lista" → `BottomSheet` nome → `PriceKeypad` prezzo
5. Bottone "Fine" → `finishSession` → naviga a /storico

### Componenti usati
- `PriceKeypad` (Vaul) — per conferma prezzo da lista e fuori lista
- `BottomSheet` (Vaul) — per nome item fuori lista e store picker
- `toast` (Sonner) — feedback dopo ogni acquisto

- `src/hooks/useItems.ts`: `useItems`, `useSupermarkets`, `useCategories`
